버억

버억은 스프링 프레임워크를 활용한 햄버거 리뷰 반응형 웹입니다.

  • 사용자들은 좋아하는 햄버거에 리뷰와 좋아요를 남길 수 있고 자유게시판을 통하여 사용자들끼리의 소통을 가능하게 합니다. 가입한 사용자들의 데이터 기반으로 나이대나 브랜드의 선호하는 햄버거를 Landbot을 이용한 챗봇으로 확인이 가능하며 추천 햄버거를 사용자에게 제공하고 있습니다. 버억은 접속자의 위치를 기반으로 주변의 햄버거 가게를 찾을 수 있는 기능을 제공합니다.

적용 기술

프레임워크 : Spring Framework

데이터베이스 : Oracle11g

적용 API : LandBot , 카카오맵 , 카카오내비

기타 : Bootstrap

  • 유튜브 링크

움직여

움직여는 스프링 프레임워크와 Teachable Machine을 활용한 홈트레이닝 반응형 웹입니다.

  • 움직여의 홈트레이닝은 Teachable Machine으로 파일을 받아 웹캠 형식으로 사용자의 움직임을 분석합니다. 비회원은 홈트레이닝만 할 수 있고 회원은 홈트레이닝과 자신이 목표개수를 설정하여 운동할 수 있으며 그 결과를 기록하여 사용자에게 제공합니다.

적용기술

프레임워크 : Spring Framework

데이터베이스 : MySQL , MongoDB

기타 : Bootstrap , Teachable Machine , AWS , SSL(HTTPS)

비밀번호 SHA256 해시함수 적용 , Jquery v2.2.4 , AWS EC2
